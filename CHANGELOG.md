# Stardust Clicker — 변경 이력

## 2026-05-22 (최신)

### 카드·등급·진행

- **신화(Mythic)** 등급 추가 (레벨업 확률 0.5%, 전설 4.5%)
- **성장의 빛**: 독립 카드 제거 → 레벨업 카드에 약 **12%** 확률로 내장, 선택 시 버프 + 무작위 보유 카드 1장 승급(신화까지)
- **하드 환생**: 레벨·XP·카드 인벤토리 초기화 (상점·DM 등은 유지)
- **우주 보존 저장소**: 최대 5장 지정, 환생 후에도 유지 (`preservedCardUids` 저장)
- **카드 합성**: 우주 개척 도감 → 카드 합성 탭 (일반2→희귀1, 희귀2→영웅1, 영웅3→전설1, 전설3→신화1)
- **스타 브레이크**: 즉시 보상만 지급, 도감(인벤토리)에 영구 보관하지 않음

### 피버 시스템

- 발동: **황금 운석** → **클릭 100회** 누적 시 **10초** 피버 (기본 지속)
- **운석 자석** 삭제 → **피버 타임 쿨타임 감소** (필요 클릭 −10/20/35/60/99)
- **운석 효과 증가** → **피버 공명 증폭** (피버 배율 %)
- **피버 연장**: 초 단위 → **지속 시간 +%** (20/40/70/120/200%, 최대 90초 캡)

### 스탯·밸런스

| 항목 | 계산 방식 |
|------|-----------|
| 클릭 %, 자동생산 %, XP % | 카드끼리 **곱연산** (`factor` 누적) |
| 크리티컬 확률 | 안테나(합) + 치명적 타격 카드(합), 판정은 **최대 100%** |
| 100% 초과 확률 | 초과분을 **크리 배율에 곱** (예: 120% → 배율 ×1.2) |
| 크리티컬 배율 | 기본+상점+카드(합) × 오버확률 배수 |

- **과부하 클릭 / 우주 가속도** (전설): **80%** (곱연산 기준 밸런스)
- **신화** 퍼센트 계열: **140%** 등으로 하향 조정 (곱연산 폭주 완화)
- **치명적 타격**: 확률·배율 수치 전반 완화

### 사운드·연출

- 첫 터치/클릭 시 BGM **페이드인** + 무한 반복 (Web Audio)
- SFX **보이스 풀** (연타 끊김 완화)
- 레벨업·카드 선택: 등급별 사운드 + 코스믹 파티클
- 카드 합성: 0.5초 흡입 연출 + 임팩트 팝업·사운드

### 저장 (localStorage)

- 키: `stardust-clicker-save-v2`
- 카드 인벤토리, 보존 슬롯 UID, 피버 클릭 누적(`clicksTowardFever`) 등 저장/복원
