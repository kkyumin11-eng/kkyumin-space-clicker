## 2026-05-22

### 카드·등급·진행

- **신화(Mythic)** 등급 추가 (레벨업 확률 0.5%, 전설 4.5%)
- **성장의 빛**: 레벨업 카드에 약 **12%** 확률로 내장, 선택 시 버프 + 무작위 보유 카드 1장 승급(영웅까지)
- **블랙홀 환생**: 레벨·XP·카드 인벤토리 초기화 (보존 저장소·DM 연구 등은 유지)
- **우주 보존 저장소**: 지정 카드 환생 후에도 유지 (`preservedCardUids` 저장)
- **카드 합성**: 일반2→희귀1, 희귀3→영웅1, 영웅4→전설1, 전설5→신화1
- **스타 브레이크**: 즉시 보상만 지급, 도감에 영구 보관하지 않음

### 피버 시스템

- 발동: **클릭 100회** 누적 또는 **황금 운석** 클릭 → **10초** 피버 (기본)
- **피버 타임 쿨타임 감소** (필요 클릭 −%)
- **피버 공명 증폭** (피버 배율 +%)
- **피버 연장**: 지속 시간 +% (최대 90초)

### 스탯·밸런스

| 항목 | 계산 방식 |
|------|-----------|
| 클릭 획득 | `(기본 + 스타더스트 주입) × 카드% × 피버 × DM 곱배율` |
| XP % | 카드끼리 **곱연산** |
| 크리티컬 확률 | 상점 + 카드(합), 판정 **최대 100%** |
| 100% 초과 확률 | 초과분을 **크리 배율에 곱** |
| 크리티컬 배율 | 기본+상점+카드(합) × 오버확률 배수 |
