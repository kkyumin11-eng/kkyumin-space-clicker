# Stardust Clicker

우주 테마 클리커 게임 (HTML / CSS / JavaScript, 빌드 없이 브라우저에서 실행).

## 실행 방법

1. `index.html`을 브라우저로 엽니다.
2. **첫 클릭(또는 화면 터치)** 후 BGM이 페이드인됩니다.
3. 진행은 `localStorage` (`stardust-clicker-save-v2`)에 자동 저장됩니다.

## 프로젝트 파일

| 파일 | 용도 |
|------|------|
| `index.html` | 게임 진입점 |
| `script.js` | 게임 로직 |
| `style.css` | 스타일·연출 |
| `CHANGELOG.md` | 변경 이력 |
| `.cursor/` | Cursor 에디터 설정 (게임 실행과 무관) |

## 주요 시스템

### 레벨업·카드

- 레벨업 시 카드 3장 중 1장 선택 → **우주 개척 도감**에 영구 누적
- 등급: 일반 → 희귀 → 영웅 → 전설 → **신화**
- 일부 카드에 **성장의 빛**(보유 카드 1장 승급, 영웅까지) 내장
- **스타 브레이크**는 즉시 스타더스트만 지급 (도감 미보관)

### 우주 개척 도감

- **보유 카드**: 클릭으로 보존 저장소 지정 (기본 1슬롯, DM 연구로 최대 10)
- **카드 합성**: 동일 종류·동일 등급 → 상위 등급 (일반2, 희귀3, 영웅4, 전설5)

### 일반 강화 상점

- **스타더스트 주입** — 클릭 고정 +2
- **광자 학습기** — 클릭 XP +10
- **확률 렌즈** / **초신성 코어** — 크리티컬 확률·배율

### 블랙홀 환생

- Stardust **1,000만** 이상 시 환생 → Dark Matter 획득
- 레벨·XP·일반 카드는 초기화, **보존 저장소** 카드만 유지

### 피버 타임

- 비피버 상태에서 **100회 클릭** 시 발동 (기본 **10초**)
- **황금 운석** 클릭 시에도 즉시 발동
- 카드·DM 연구로 지속 시간·배율·필요 클릭 수 조절 가능

### 스탯 계산 요약

- **클릭 획득**: `(기본 + 스타더스트 주입) × 카드% × 피버 × DM 곱배율`
- **XP %**: 카드 효과 **곱연산**
- **크리티컬 확률**: 상점 + 카드 **합연산**, 상한 100%
- **100% 초과 확률**: 크리티컬 **배율에 곱** (예: 120% → ×1.2)
