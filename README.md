# Stardust Clicker

우주 테마 클리커 게임 (HTML / CSS / JavaScript, 빌드 없이 브라우저에서 실행).

## 실행 방법

1. `index.html`을 브라우저로 엽니다.
2. **첫 클릭(또는 화면 터치)** 후 BGM이 페이드인됩니다.
3. 진행은 `localStorage`에 자동 저장됩니다.

## 주요 시스템 (현재 버전)

### 레벨업·카드

- 레벨업 시 카드 3장 중 1장 선택
- 등급: 일반 → 희귀 → 영웅 → 전설 → **신화**
- 일부 카드에 **성장의 빛**(랜덤 카드 1장 승급) 내장
- **스타 브레이크**는 즉시 스타더스트만 지급 (도감 미보관)

### 우주 개척 도감

- **보유 카드**: 클릭으로 보존 저장소(5슬롯) 지정
- **카드 합성**: 동일 종류·동일 등급 → 상위 등급 1장

### 블랙홀 환생

- Stardust 100,000 이상 시 환생 → Dark Matter 획득
- 레벨·XP·일반 카드는 초기화, **보존 저장소 5장**만 유지

### 피버 타임

- 비피버 상태에서 **100회 클릭** 시 발동 (기본 **10초**, 카드로 지속 %·필요 클릭 감소 가능)
- 스탯 패널에서 **피버 충전 진행도** 확인

### 스탯 계산 요약

- **클릭 % / 자동생산 % / XP %**: 카드 효과 **곱연산**
- **크리티컬 확률**: 안테나 + 카드 **합연산**, 상한 100%
- **확률 100% 초과분**: 크리티컬 **배율에 곱** (예: 120% → ×1.2)
